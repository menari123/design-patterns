# Design Patterns

Este repositório contém estudos e exemplos práticos de Design Patterns.

## Objetivo

Explorar e documentar os principais padrões de design utilizados no desenvolvimento de software, com exemplos de implementação e casos de uso.

## Estrutura

- `/creational` - Padrões de criação
- `/structural` - Padrões estruturais
- `/behavioral` - Padrões comportamentais

## Como usar

Cada pasta contém exemplos comentados de implementação dos padrões.
