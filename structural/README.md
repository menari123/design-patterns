# Padrões Estruturais

Esta pasta contém exemplos e documentação sobre **padrões de design estruturais**.

Os padrões estruturais tratam da composição de classes e objetos, facilitando a criação de estruturas flexíveis e reutilizáveis.

## Padrões Inclusos

- Adapter
- Bridge
- Composite
- Decorator
- Facade
- Flyweight
- Proxy

Para mais informações sobre cada padrão, consulte as subpastas correspondentes.
