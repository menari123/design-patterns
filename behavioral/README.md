# Padrões Comportamentais

Esta pasta contém exemplos e documentação sobre **padrões de design comportamentais**.

Os padrões comportamentais focam em comunicação entre objetos, definindo como eles interagem e distribuem responsabilidades.

## Padrões Inclusos

- Chain of Responsibility
- Command
- Interpreter
- Iterator
- Mediator
- Memento
- Observer
- State
- Strategy
- Template Method
- Visitor

Para mais informações sobre cada padrão, consulte as subpastas correspondentes.
