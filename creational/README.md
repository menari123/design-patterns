# Padrões Criacionais

Esta pasta contém exemplos e documentação sobre **padrões de design criacionais**.

Os padrões criacionais focam na criação de objetos, definindo mecanismos que promovem reutilização de código e flexibilidade na instanciação.

## Padrões Inclusos

- Abstract Factory
- Builder
- Factory Method
- Prototype
- Singleton

Para mais informações sobre cada padrão, consulte as subpastas correspondentes.
